AbleAssist

AbleAssist is an accessibility-focused web application designed to make digital interaction more inclusive for everyone. Our project combines voice assistance, sign language recognition, and multimodal interaction to help bridge communication gaps for people with different abilities.

The core idea behind AbleAssist is to allow users to interact naturally‚Äîwhether by speaking, typing, or using hand gestures‚Äîwithout barriers. By integrating real-time camera input, speech recognition, and gesture-based controls, we aim to create a platform that adapts to the user, not the other way around.

This project was developed with a strong focus on:

  • Accessibility and inclusivity
  • Real-time voice and sign language interaction
  • User-friendly and intuitive design
  • Assistive technology for everyday use

AbleAssist is built as a learning-driven project, combining web technologies and machine learning concepts to explore how AI can be used responsibly to improve accessibility.

How to use?

  • Open the website using a modern browser (Chrome recommended).
  • Ensure you have a working camera and internet connection.
  • When prompted, allow camera access.
  • Turn Camera ON to start gesture detection.
  • Show a trained hand gesture clearly in front of the camera.
  • The detected gesture will appear automatically on screen.
  • Normal users do not need to train gestures.
  • Gestures are already trained and shared across all devices.

To add new gestures (admin/trainer only):

  • Turn Camera ON.
  • Select the gesture label.
  • Turn Training Mode ON.
  • Show the gesture clearly for 2-3 seconds.
  • Training count will increase automatically.
  • Turn Training Mode OFF after training.
  • Newly trained gestures are saved and available on all devices.
  • Open the same website link on another phone or laptop.
  • Previously trained gestures will work without retraining.

On the multimode page:

  • Gestures trigger actions automatically.
  • Use the search bar to type commands (example: youtube).
  • Press Enter to perform the action.

For best results:

  • Use good lighting.
  • Keep your hand fully visible.
  • Avoid fast hand movement while training.

If something doesn't work:

  • Refresh the page.
  • Check camera permission is set to Allow.
  • Turn camera OFF and ON again.
  • Ensure internet connection is active.
  • Training saves multiple frames per gesture -- this is normal and improves accuracy.
  • Do not close the browser tab while training is in progress.
