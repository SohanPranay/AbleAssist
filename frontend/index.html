AbleAssist

AbleAssist is an accessibility-focused web application designed to make digital interaction more inclusive for everyone. Our project combines voice assistance, sign language recognition, and multimodal interaction to help bridge communication gaps for people with different abilities.

The core idea behind AbleAssist is to allow users to interact naturally‚Äîwhether by speaking, typing, or using hand gestures‚Äîwithout barriers. By integrating real-time camera input, speech recognition, and gesture-based controls, we aim to create a platform that adapts to the user, not the other way around.

This project was developed with a strong focus on:

  • Accessibility and inclusivity
  • Real-time voice and sign language interaction
  • User-friendly and intuitive design
  • Assistive technology for everyday use

AbleAssist is built as a learning-driven project, combining web technologies and machine learning concepts to explore how AI can be used responsibly to improve accessibility.
